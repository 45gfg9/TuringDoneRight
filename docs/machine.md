## 电脑

### 基本知识

电脑想要跑（<span class = "heimu" title = "冷笑话？">~~那也跑不了~~</span>），有几个比较重要的组成部分（同时也是选购电脑时需要重点考虑的一些对象）：

??? general "CPU"

    `Intel 11th`<sub>旧</sub> `Intel 12th`<sub>新</sub> `AMD 5000 series`<sub>旧</sub> `AMD 6000 series`<sub>新</sub>  
    除去各种各样的 低压U 之外（如 `I7-1280P` <span class = "heimu" title = "呜呜">其实是我弄不懂</span>），个人认为可以考虑的型号：

        `I5-12500H`<sub>新</sub>  
        `I7-12700H`<sub>新</sub>
    === "AMD"
        `R5-5600H`<sub>旧</sub>  
        `R7-5800H`<sub>旧</sub>  
        `R5-6600H`<sub>新</sub>  
        `R7-6800H`<sub>新</sub>
    === "APPLE"

??? general "内存"
    目前主流内存包括 `DDR4`<sub>旧</sub> 与 `DDR5`<sub>新</sub> 两种。后者因为市场价格与开发适配问题尚未广泛推广，目前主要是在 `Intel 12th` `AMD 6000 series` 这些新平台上使用，频率也尚未提高，目前跟 `DDR4` 没有太大的差别。  
    所以考虑的时候主要还是考虑内存的大小。主流的笔记本内存大小一般为 `16G` 且 **足够** 日常使用。少数为 8G 。若有对较大的对内存的需求（如视频剪辑），可以考虑更高的内存容量。

??? general "外存"
    外置存储器，~~也就是常说的内存~~。目前笔记本内外置存储器以固态硬盘为主，笔记本内机械硬盘近年来应该是绝迹状态。  
    根据传输模式而言，目前包括两大类： `PCI-E 3.0` `PCI-E 4.0`。除了 `AMD 5000 series` 不支持 `PCI-E 4.0` 之外，`Intel 11/12th` `AMD 6000 series`均支持这两种模式。
    对于一般配置， `512G` 的存储是较为常见的，不过不一定够用。我目前是 `512G + 1T` 的状态 <font class = "heimu" title = "喵..?" size = 1>~~才不是因为下了很多游戏然后没玩也没删~~</font> ，感觉比较舒适。不过一开始时不一定有必要纠结外置存储的大小，例如图灵每学期一次的报销款可以用来买硬盘。如果电脑的拓展性足够，借报销的机会购买硬盘来拓展外置存储大小也是一种选择<font class = "heimu" title = "我不是果黑(×)" size = 1>在这里想 diss 苹果，详情请搜索或者随便抓个科技宅问问（×）</font>。  
    > 购买提示：  
    > ① 建议看品牌，三星、西数、致态、铠侠等。如果不是迫不得已或者对性能安全没什么需求，尽量不要无脑冲最低。  
    > ② 因为我的电脑不支持 PCI-E 4.0 ，没尝试过 PCI-E 4.0 的硬盘，不过好像说虽然 PCI-E 4.0 比 PCI-E 3.0 读写速度都高了很多（例如读取速度，前者 : 后者 ≈ 7000 MB/s : 3500 MB/s）在日常使用的时候差别并不是很明显。  
    > ③ 此外，最近的无缓方案感觉尚可。我的 980 用着没什么问题（大概）。  
    > **④ 我的说法可能有些问题，具体的可以参考 B站 等平台的评测信息。**

除了以上的因素，还有一些因素值得考虑：

??? general "电源"
    我这里提的电源更多的指代电池容量（当然不是说电源适配器不重要，<font class = "heimu" title = "冷咖啡离开了杯垫，我忍住的情绪在很后面" size = 1>比如我 ~~巨坑~~ 的 天选2 ，适配器功率不够，高负载下还会从电池偷电来着</font>）。电池容量很大程度上与续航成正相关，一般来说电池容量越大越利于电脑的续航时间延长。增加的续航时间可以延长 ~~我们的生命~~ 。当然除了电池容量之外 CPU 等器件的功耗也相当重要。例如游戏本如果让独显工作，续航时间会大幅缩短。  
    一般来说电池容量 `50-90 Wh` 比较常见。一些轻薄本虽然电池容量在 `50-70 Wh` ，但由于整体功耗低，续航时间也可能大于一些电池容量达到 `90 Wh` 的游戏本。

??? general "屏幕"
    === "分辨率"
        整理资料 <font class="heimu" title = "喵呜？" size = 1>水 B站</font> 的时候发现有些人主张“今年还不上 2k 不能接受”。我个人觉得虽然 `2k` 分辨率( 常规 `2560*1440` )的屏幕的确有提升，但也不是说 `1080P` ( 常规 `1920*1080`)的屏幕不能使用（×）。就观感而言，如果不是对画面特别敏感或者有特别的需求，一般来说差别应该不是特别大。（这边提到的像素比例都是按 `16:9` 说的）  
    === "屏幕比例与大小"
        近年来比较火的屏幕比例是 `16:10` ，的确这样子的屏幕看起来占比更大更舒服。看个人喜好选择吧。  
        对于屏幕大小，个人感觉 `15.6 英寸` 算是一个分界岭。小于会显得小巧，大于会显得有些笨重。 <font class = "heimu" title = "不听不听" size = 1> ~~也有可能是我用惯了 15.6 英寸~~ </font>  
    === "屏幕亮度"
        屏幕亮度决定了在亮光环境下电脑屏幕是否能够被清晰的看见，与 `HDR` 也有一定的关系。 <font class = "heimu" title = "你知道的太多了" size = 1>~~但是一般不会有人特意把电脑搬到太阳底下用吧（×）~~</font> `300-400 nit` 比较常见，也应当够用。  
    === "色域 与 色深"
        此外对于色域这方面，我了解的不算多，但是色域越高色彩越饱满。常见的较高色域包括 `100% sRGB` `95% DIC-P3` `72% NTSC` 之类的。可能说尽量不要接触色域明显低于前面几者的机器吧？  
        色深常见 `8 bit` `10 bit` ， `10 bit` 可能画面表现力上会更好，但是我没怎么看出来...？可以说如果有 `10 bit` 那最好，但是只有 `8 bit` 也没什么问题。  
    === "刷新率 与 响应速度"
        对于游戏玩家，屏幕刷新率可能比较重要。一般而言，轻薄办公本屏幕刷新率在 `60 / 90 Hz` ，游戏本常见的刷新率有 `120 / 144 / 165 / 240 / ... Hz` 。个人而言，大于等于 `240 Hz` 的刷新率可能在观感上没有与低于它们的屏幕没有什么差别。根据自己的需求选择即可。此外，屏幕响应时间也可以看看。<font class = "heimu" title = "你知道的太多了" size = 1>不过不知道为什么我感觉大家宣传的都是 3ms （×）</font> 。
    === "其他"

        === "防蓝光认证"
            为了自己的眼睛着想，~~可以选择具有防蓝光认证的机器~~。
        === "OLED"  
            就目前而言，很多人不是特别推荐 `OLED` 的屏幕，说容易烧屏。虽然我感觉可能问题不大？（因为现在很多旗舰手机和高端电视都在使用 `OLED` 的屏幕）  
        === "触屏功能"
            触屏功能仁者见仁智者见智。<font class = "heimu" title = "你知道的太多了呜呜" size = 1>我一开始是想要一本能写的轻薄本的，直到被父母的朋友堵了。</font> 不过如果你还想配平板的话大概是不需要的。

??? general "接口"
    对于一台健全的笔记本，足够使用的接口不可缺少。目前常见的接口有：  
    > `USB-A` `USB-C`(即 `TYPE-C`) `HDMI` `DisplayPort` `RJ45` `极为先进的 3.5mm 耳机接口` `SD卡槽` `Lightning`  

    一般来说，对于全能本、游戏本而言这些接口不会少。值得注意的是轻薄本这一类笔记本可能会为了机身的小巧而砍掉一大部分的接口，比如 `RJ45` 。对于少接口的笔记本，可以考虑使用拓展坞。  
    除去机身外的接口，机身内部的接口也值得注意。例如厂商是否预留了第二块 `M.2` 接口（用于拓展硬盘），是否具有可更换的内存。

??? general "键盘和触摸板"
    键盘我觉得习惯就行（~~实在不行买外设~~）。比较让人在意的方面有：方向键是否是全尺寸的、是否有小键盘等。  
    触摸板的话也要看个人习惯。像我是触摸板重度依赖症，比较喜欢大尺寸的一体化触摸板 <font class = "heimu" title = "我不忍住，落泪感觉" size = 1>虽然我的 天选2 的触摸板...</font> ，但是有些人就喜欢禁用触摸板。  

??? general "其他功能"

    === "独显直连"
        可以提高游戏帧数、 ~~增加耗电~~ 。
    === "PD 充电"

    其他比较小众的有 稀奇古怪的旋钮（ 如华硕 ProArt 创16 ） 和 虚拟工具栏（ 如21款 Macbook Pro ）。

??? general "内存 和 外存"
    对于手机内存而言，主要是 `LPDDR4(X)` `LPDDR5(X)` 。但是在当今二者差别应该不是特别大，所以厂商宣传的主要还是内存容量，例如 `8G` `12G` ；对于外置存储，通常有 `128G` `256G` `512G` 。组在一起，也就有了常说的 `8 + 256`  `12 + 512` 。对我来说， `8 + 256` 在我的生活中没有产生什么阻碍，但也可能在未来有所不足。  
    此外，有一个概念称为 `UFS` ( Universal Flash Storage )，亦称为 `通用闪存存储` 。一般来说，版本越高，数据传输越快。详情可以参考 [维基百科](https://zh.wikipedia.org/zh-cn/%E9%80%9A%E7%94%A8%E9%97%AA%E5%AD%98%E5%AD%98%E5%82%A8) 。  
    此外也有诸如 智慧运存（荣耀华为） 、 拓展运存 之类的操作，将存储空间（外存）转化为可用的内存使用。

??? general "电池 与 快充"
    电池容量是手机上相当重要的一个组成部分，对于手机的续航具有重要意义。常见的电池容量包括 `4000 ~ 5000 mAh` 。 <font class = "heimu" title = "为发烧而生" size = 1>当然对于某些火龙来说，无论容量有多大都是浮云。</font>  
    如果电池容量的确不大，那还有什么能拯救我们呢？那就是快充。快充瓦数从 `20 ~ 150 W` 不等。一般来说 `65 W` 快充的机型偏多，大概能做到 `30 ~ 40 min` 充满的水平。对我来说属于够用的水平。

??? general "镜头 与 相机"
    镜头与相机调教是 中低端机型 和 高端机型 比较重要的分界岭。就我的认知而言，一般主打性价比的手机镜头素质和相机调教做的不一定好，~~大多数属于 “ 扫个码 ” 的水平~~；而较高端的手机则会在这方面有所侧重。

??? general "屏幕"
    与电脑那一块类似，这里的屏幕也包括以下内容：  
    === "分辨率"
        对于手机屏幕而言，个人感觉 `1080P` 和 `2K` 差别应该不是很大，很多屏幕素质达到 `2K` 的手机在动态分辨率等设置上也会设定在某些场景下降低分辨率来节约电量。如果是追求性价比，那么 `1080P` 未尝不是一种选择。  
    === "屏幕比例和尺寸"
        我总感觉手机的屏幕比例各种奇怪，特别是在某些厂商的曲面屏上。对应全屏拍照的比例也会受到影响。 ~~不过一般拍照我还是喜欢用 `4:3` 的比例~~ 不过一般来说影响不大，现在很多软件也做了异形屏的适配。  
        屏幕尺寸方面，看个人习惯和个人特征。<font class = "heimu" title = "喵！喵！！" size = 1> ~~比如我手小，喜欢比较小巧的手机 比如6.1英寸的LG G8~~ </font>  
    === "屏幕亮度"
        手机的屏幕平均亮度在 `300-500 nit` 比较常见，而厂商在发布会上说的 `1000+ nit` 一般指代 `局部峰值亮度` 。这个越高，在阳光下看手机屏幕会越清晰。 <font class = "heimu" title = "反转了喵！" size = 1>应该不会有人不在阳光下看手机吧？ ~~虽然超烫~~ </font>  
    === "色域 与 色深"
        色域和色深问题同电脑，但是在手机这方面好像卷的比较少？  
    === "刷新率 与 响应速度"
        现在的手机基本都会向 高刷（新率） 看齐。一般而言，多数手机刷新率在 `90 ~ 144 Hz` ，同时也会做到可变刷新率。 `120 Hz` 左右应当足够使用。  
        响应速度好像一般不谈。  
    === "其他"
        === "OLED"  
            手机和电脑市场不同，近期 `OLED` 屏幕在手机市场有比较大的比重。~~感觉看过去全是 `OLED` ？~~
        === "屏下指纹"
            包括 `光学屏下指纹` 和 `超声波屏下指纹` ，虽然我这个门外汉没去了解过两者的差别。
        === "前置摄像头"
            终于要提到 ~~罪大恶极的打孔摄像头~~ 了。目前很多手机厂商的前置摄像头方案都是设置打孔摄像头，在你的屏幕上留下一块黑洞；尚未广泛采用的方案还有采用隐藏式屏下摄像头，通过降低屏幕某一区域的像素比例来让其下的摄像头获取到外界的光，从而在实现整块屏幕显示的同时实现前置摄像头的功能。<font class = "heimu" title = "呜呜" size = 1>虽然我不知道为什么同样能实现全屏幕且更加安全的弹出式摄像头不能被厂商沿用。</font>

## 平板

!!! warning "平板并不是必须"
    如果你只是为了做笔记，尽管平板记录的效率会大于纸笔，但是如果你的学习习惯是纸笔，那么还是不建议你为了平板而平板。更何况再添置平板也是一种负担。

### 补充知识

与手机基本类似，但是有几个需要特别提及的点。  

=== "散热"
    对于平板而言，面积大，散热相对手机可能会好一点。~~这时候上火龙或许也不是不行。~~  
=== "电池 和 快充"
    平板的电池容量比手机要大，有的甚至能达到 `10000+ mAh` ，但是这也不意味着你能把平板当作充电宝，小心双双没电。 <font class = "heimu" title = "同是天涯沦落人" size = 1> ~~问就是亲身经历~~ </font> 此外，平板的快充一般比不上手机，充电的时候注意预留时间。  
=== "屏幕刷新率"
    平板屏幕的刷新率比较重要： `120 Hz` 的刷新率比起 `60 Hz` 的刷新率能够让手写的时候体验更加丝滑，~~也能让舍友的 `MatePad 11` 当场羞辱我的 `MatePad Pro 10.8`~~ 。  
=== "手写"
    如果平板确实看重手写，可能你得特别关注一些手写优化比较好的品牌，例如 苹果 之类。  
    各品牌手写相关的情况可以在各平台上查询。
   
!!! note "提醒"
    苹果在每年 `7 ~ 9月` 有 返校季活动 ，凭借教育优惠可以嫖耳机。如果对 苹果 感兴趣的话可以关注这活动的相关通知。
